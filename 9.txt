

1) START CDH & HUE SERVICES
---------------------------
• Boot Cloudera VM (Cloudera Quickstart VM)  
• Start Cloudera Manager (if not running)  
• Make sure the following services are running:
  - HDFS
  - YARN
  - Hive
  - Hue
  - Impala (optional)
  
2) ACCESS HUE (WEB UI)
----------------------
Open browser:
http://localhost:8888

Default login:
username: cloudera
password: cloudera

3) UPLOAD SAMPLE DATASET
------------------------
In Hue:
File Browser → Upload

Upload **employee.csv**:
id,name,dept,salary
1,Arvind,IT,50000
2,Saba,HR,60000
3,Jayashree,IT,55000
4,Shilpa,Marketing,85000

Path after upload will be:
 /user/cloudera/employee.csv

4) CREATE HIVE TABLE (IN HUE)
-----------------------------
Go to:
Query Editors → Hive

Run:

CREATE TABLE IF NOT EXISTS employee (
  id INT,
  name STRING,
  dept STRING,
  salary INT
)
ROW FORMAT DELIMITED
FIELDS TERMINATED BY ','
STORED AS TEXTFILE;

Load the CSV into Hive:

LOAD DATA INPATH '/user/cloudera/employee.csv' INTO TABLE employee;

5) ANALYZE DATA USING HIVE QUERIES
----------------------------------
a) Count employees in each department:
SELECT dept, COUNT(*) FROM employee GROUP BY dept;

b) Average salary per department:
SELECT dept, AVG(salary) FROM employee GROUP BY dept;

c) Maximum salary:
SELECT MAX(salary) FROM employee;

6) GENERATE REPORTS IN HUE
--------------------------
• Go to Dashboards / Charts  
• Choose a Hive query result  
• Select Chart type:
  Bar / Pie / Line / Table  
• Save and Export report (PDF/Image)

FINAL RESULT
------------
You can now visualize:
• Department-wise employee count  
• Average salary distribution  
• Maximum salary  
• Customized dashboards and reports  
using Hue’s visual interface (no coding required).
